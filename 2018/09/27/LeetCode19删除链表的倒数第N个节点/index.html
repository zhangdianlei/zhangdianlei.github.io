LeetCode19删除链表的倒数第N个节点

发表于 2018-09-27 | 更新于 2018-12-21 | 分类于 算法 |

题目描述：

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

说明：

给定的 n 保证是有效的。

解题思路：

大体想到了两个思路：

  1. 先说暴力的方法。先遍历一遍链表，可以得知总计有多少的节点。删除倒数第n个，就等同于删除正数第 $len(l) - n + 1$ 个。遍历两次链表就肯定可以解决。函数复杂度也就是$O(n)$

  2. 想用遍历一次链表就实现该功能的话，就需要使用两个指针。一个指针先行一步，当前一个指针走了n步之后，后一个指针再动身。这样，等第一个指针走到最后的时候，后面的指针跳过其下一个节点，就表示删除了该节点了。这个方法不难理解，关键就是需要能够对链表及“指针”有着较好的理解，能够处理好边界条件。
